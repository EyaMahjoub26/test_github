# Gestion des Étudiants (CRUD) - Dark Mode & Glassmorphism

## Description du projet
Ce projet est une application web permettant de gérer des étudiants de manière simple et efficace. Elle offre la possibilité d’ajouter, modifier, supprimer, rechercher et trier  des étudiants tout en proposant une interface moderne avec **Glassmorphism** . L’objectif est de créer une expérience utilisateur agréable et intuitive sur tous les types d’écrans.

## Technologies utilisées
- **HTML5** : structure de la page et des formulaires  
- **CSS3** : design, responsive, effets Glassmorphism et dark mode  
- **JavaScript** : manipulation du DOM et gestion des actions CRUD  
- **Git & GitHub** : versionning et déploiement avec GitHub Pages  

## Fonctionnalités principales
- Ajouter un étudiant avec son nom, prénom, âge et photo  
- Modifier les informations d’un étudiant existant  
- Supprimer un étudiant  
- Rechercher des étudiants en temps réel  
- Trier les étudiants par ordre alphabetique, par age ou par filiere 
- Affichage des données sous forme de grille responsive 
- Basculer entre mode sombre et claire

## Nouveautés explorées
Ce projet m’a permis de renforcer mes compétences en **manipulation dynamique du DOM**, en gestion de formulaires et en design moderne. J’ai découvert comment appliquer le style Glassmorphism de manière cohérente. J’ai également amélioré mes connaissances en responsive design et en structuration de code pour un projet fonctionnel et maintenable.

## Difficultés rencontrées
Le projet a présenté des défis liés à la gestion des données et à l’interface. Il a fallu trouver un équilibre entre design et fonctionnalité, notamment pour que l’interface reste intuitive, réactive et esthétique sur tous les types d’écrans. L’organisation du code pour rendre les différentes fonctionnalités CRUD fiables et faciles à maintenir a également demandé réflexion et ajustements notament l'ajout dinamique des grilles d'etudiants.

## Solutions apportées
Pour surmonter ces difficultés, j’ai adopté une approche progressive et méthodique. Le code a été structuré de manière claire et modulable, permettant une maintenance simple et une meilleure lisibilité. Le design a été fait avec des ajustements CSS pour le Glassmorphism et le mode sombre. Les fonctionnalités ont été testées de manière régulière afin de garantir un comportement stable et une expérience utilisateur optimale.
